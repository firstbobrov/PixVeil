Руководство пользователя

Подробная инструкция по использованию сервиса PixVeil

Возможности сервиса

Скрытие текстовых сообщений внутри изображений
Извлечение сообщений из зашифрованных изображений
Поддержка нескольких алгоритмов шифрования
Дополнительная защита с помощью секретных ключей

Шифрование данных

Выберите режим и алгоритм

Переключитесь в режим шифрования и выберите подходящий алгоритм из списка. Для большинства случаев подойдет алгоритм "Базовый".

Загрузите изображение

Нажмите кнопку выбора файла и загрузите изображение в формате PNG или JPG. Рекомендуется использовать изображения с высоким разрешением.

Укажите секретный ключ (опционально)

Если хотите усилить защиту, введите секретный ключ. При дешифровании потребуется указать этот же ключ.

Введите сообщение

Напишите текст, который нужно зашифровать. Максимальная длина зависит от размера изображения и выбранного алгоритма.

Получите результат

После обработки вы увидите зашифрованное изображение, которое можно скачать или скопировать для передачи получателю.

Дешифрование данных

Выберите режим и алгоритм

Переключитесь в режим дешифрования. Убедитесь, что выбран тот же алгоритм, который использовался при шифровании.

Загрузите изображение

Загрузите зашифрованное изображение, полученное от отправителя. Убедитесь, что файл не был изменен после шифрования.

Укажите ключ (если требуется)

Если при шифровании использовался секретный ключ, введите его в соответствующее поле.

Получите сообщение

После обработки вы увидите исходное сообщение, которое было зашифровано в изображении.

Какой алгоритм выбрать?

Базовый

✓ Быстрое шифрование

✓ Размер файла почти не меняется

✓ Для коротких сообщений

Модифицированный

✓ Намного больше вместимости

⚠ Шифрование дольше, файл тяжелее

✓ Идеален для длинных текстов

Совет: для коротких заметок — базовый алгоритм, для длинных сообщений — модифицированный.

Важные замечания

Сервис поддерживает только изображения в форматах PNG и JPG
Качество шифрования зависит от размера и сложности изображения
Секретный ключ значительно повышает безопасность зашифрованных данных
Всегда сохраняйте оригинальное изображение перед шифрованием
