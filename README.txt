PixVeil

PixVeil — это проект для стеганографии и шифрования сообщений в изображениях. Программа позволяет скрывать текстовые сообщения внутри картинок (JPG/PNG) с использованием различных алгоритмов шифрования, а также извлекать их обратно.

Основной функционал

- Режим шифрования: скрытие текста внутри изображения с опциональным ключом шифрования.
- Режим дешифрования: извлечение скрытого сообщения из изображения.
- Поддержка разных алгоритмов шифрования и форматов изображений.
- Работа с пользовательскими и шаблонными изображениями.

Список существующих скриптов:

png_to_jpg.py
Конвертирует PNG-изображения в формат JPG с сохранением в папку Temp.

jpg_to_png.py
Конвертирует JPG-изображения в формат PNG с сохранением в папку Temp.

image_to_RGBmatrix.py
Загружает изображение и преобразует его в RGB-матрицу (numpy array) для дальнейшей обработки.

RGBmatrix_to_image.py
Сохраняет RGB-матрицу обратно в изображение (JPG или PNG) в папку Temp.

encode_decode_message.py
Кодирует строку в массив индексов согласно заданному набору символов (char_set) и декодирует обратно.

shuffle_char_set.py
Генерирует перемешанный набор символов на основе числового ключа (seed). Используется для создания уникальных алфавитов шифрования.

optional_cipher(Fernet).py
Реализует шифрование и дешифрование текста с использованием алгоритма Fernet на основе пароля пользователя.

temp_clean.py
Очищает папку Temp — удаляет все или указанные файлы.

Base_Encryption.py
Собственный алгоритм стеганографического преобразования, среднее время работы, средняя вместительность.

Base_Decryption.py
соответсвующий дешифратор для алгоритма выше.

my_shifr_rabotaet_no_medleno.py
Собственный алгоритм стеганографического преобразования, работает долго, но очень вместительный

my_deshifrator
соответсвующий дешифратор для алгоритма выше.
